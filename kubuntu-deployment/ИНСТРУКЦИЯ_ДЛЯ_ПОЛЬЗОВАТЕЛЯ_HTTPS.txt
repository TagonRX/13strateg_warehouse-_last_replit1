📋 ЧТО БЫЛО СДЕЛАНО

Создан автоматический скрипт установки HTTPS для локального сервера Kubuntu.

После установки HTTPS:
  ✅ Камера телефона БУДЕТ РАБОТАТЬ
  ✅ Можно сканировать баркоды через браузер
  ✅ USB сканер продолжает работать как раньше

═══════════════════════════════════════════════════════════════════════════════

📦 НОВЫЕ ФАЙЛЫ ДЛЯ СКАЧИВАНИЯ

Скачайте эти файлы из Replit и скопируйте на сервер Kubuntu:

   1. setup-https.sh
      → Автоматический скрипт установки HTTPS
   
   2. БЫСТРАЯ_УСТАНОВКА_HTTPS.txt
      → Краткая инструкция (3 минуты)
   
   3. НАСТРОЙКА_HTTPS.txt
      → Подробная инструкция и устранение проблем

Копируйте в папку: ~/Desktop/Main-project-13strategyexperts/

═══════════════════════════════════════════════════════════════════════════════

🚀 БЫСТРАЯ УСТАНОВКА (3 КОМАНДЫ)

На сервере Kubuntu выполните:

   1. cd ~/Desktop/Main-project-13strategyexperts
   
   2. chmod +x setup-https.sh
   
   3. sudo ./setup-https.sh

Готово! Скрипт всё настроит автоматически.

═══════════════════════════════════════════════════════════════════════════════

📱 КАК ИСПОЛЬЗОВАТЬ

После установки HTTPS:

   1. На телефоне откройте: https://192.168.0.168
      
   2. Примите предупреждение о сертификате:
      "Дополнительно" → "Перейти на сайт"
      
   3. Войдите: admin / admin123
      
   4. Перейдите в "Режим сканера"
      
   5. Нажмите "Запустить камеру"
      
   6. ✅ КАМЕРА РАБОТАЕТ!

💡 ВАЖНО ЗНАТЬ

• Предупреждение о сертификате - это НОРМАЛЬНО
  (самоподписанный сертификат для локальной сети)

• Предупреждение нужно принять ОДИН РАЗ
  После этого браузер запомнит

• Теперь открывайте через https:// вместо http://

• Порт изменился: 443 вместо 5000
  (но это прозрачно, просто https://IP)

🔧 ЧТО ДЕЛАЕТ СКРИПТ

Скрипт автоматически:

   ✓ Устанавливает Nginx
   ✓ Создаёт SSL сертификат для вашего IP
   ✓ Настраивает HTTPS на порту 443
   ✓ Настраивает WebSocket для удалённого сканера
   ✓ Автоматически перенаправляет HTTP → HTTPS
   ✓ Запускает всё

Ваше приложение продолжает работать на порту 5000.
Nginx выступает как прокси-сервер между телефоном и приложением.

❓ ПРОБЛЕМЫ

Если что-то не работает:

   1. Читайте НАСТРОЙКА_HTTPS.txt
      (там раздел "Устранение проблем")
   
   2. Проверьте что Nginx запущен:
      sudo systemctl status nginx
   
   3. Проверьте что приложение работает:
      pm2 status
   
   4. Проверьте логи:
      sudo tail -f /var/log/nginx/error.log
