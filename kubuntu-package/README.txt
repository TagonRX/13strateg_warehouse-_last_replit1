📝 ВСЕ НАСТРОЙКИ В ОДНОМ ФАЙЛЕ: kubuntu-config.js

Откройте файл kubuntu-config.js чтобы изменить:
  • Пароль PostgreSQL (по умолчанию: БЕЗ пароля)
  • Порт веб-сервера (по умолчанию: 5000)
  • Путь к файлу данных
  • Название базы данных
  • Любые другие параметры

Все комментарии в файле на русском языке!


═══════════════════════════════════════════════════════════════════════
  🚀 УСТАНОВКА (ОДИН РАЗ)
═══════════════════════════════════════════════════════════════════════

1. Откройте терминал (Ctrl+Alt+T)

2. Распакуйте архив на Desktop:
   
   cd ~/Desktop
   tar -xzf warehouse.tar.gz
   cd warehouse

3. (ОПЦИОНАЛЬНО) Откройте kubuntu-config.js и измените настройки

4. Запустите установку:
   
   ./install.sh

5. Ждите 10-15 минут пока установится все


═══════════════════════════════════════════════════════════════════════
  ▶ ЗАПУСК СИСТЕМЫ
═══════════════════════════════════════════════════════════════════════

./start.sh

Откройте браузер: http://localhost:5000
Логин: admin
Пароль: admin123

💡 Если вы изменили порт в kubuntu-config.js, используйте новый порт!


═══════════════════════════════════════════════════════════════════════
  ■ ОСТАНОВКА
═══════════════════════════════════════════════════════════════════════

./stop.sh


═══════════════════════════════════════════════════════════════════════
  🗑 ПОЛНАЯ ОЧИСТКА
═══════════════════════════════════════════════════════════════════════

Если нужно все удалить и переустановить заново:

./clean.sh

После этого можно снова запустить ./install.sh

tail -f warehouse.log


═══════════════════════════════════════════════════════════════════════
  🔧 ИЗМЕНЕНИЕ НАСТРОЕК ПОСЛЕ УСТАНОВКИ
═══════════════════════════════════════════════════════════════════════

1. Остановите сервер: ./stop.sh

2. Откройте и измените: kubuntu-config.js

3. Если изменили настройки БД - переустановите:
   ./clean.sh
   ./install.sh

4. Если изменили только порт сервера - просто запустите:
   ./start.sh


═══════════════════════════════════════════════════════════════════════
  ℹ ВАЖНО
═══════════════════════════════════════════════════════════════════════

• ВСЕ НАСТРОЙКИ в файле kubuntu-config.js (с русскими комментариями)
• PostgreSQL по умолчанию БЕЗ пароля (trust mode)
• Если PostgreSQL требует пароль - измените в kubuntu-config.js
• Все 72,439 записей загружаются автоматически при установке
• Пользователь admin создается автоматически
• Система работает только на этом компьютере (localhost)


═══════════════════════════════════════════════════════════════════════
  ⚠ ЕСЛИ НЕ НАХОДИТ ФАЙЛЫ ИЛИ БАЗУ ДАННЫХ
═══════════════════════════════════════════════════════════════════════

Откройте kubuntu-config.js и проверьте:

  • paths.dataFile - правильный путь к database-export.json?
  • database.host - правильный адрес PostgreSQL?
  • database.password - нужен ли пароль?

Все пути и настройки можно изменить в этом файле!


═══════════════════════════════════════════════════════════════════════
  ⚠ ЕСЛИ ПРОБЛЕМЫ
═══════════════════════════════════════════════════════════════════════

2. Полная очистка: ./clean.sh

3. Проверьте kubuntu-config.js

4. Переустановка: ./install.sh

5. Запуск: ./start.sh
